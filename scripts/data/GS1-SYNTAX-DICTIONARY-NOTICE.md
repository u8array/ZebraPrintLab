# GS1 Barcode Syntax Dictionary

The AI catalog data in the app bundle (`src/lib/gs1AiCatalog.ts`, generated by
`scripts/gen-gs1-catalog.mjs`) is derived from the GS1 Barcode Syntax
Dictionary, vendored verbatim at `scripts/data/gs1-syntax-dictionary.txt`.
