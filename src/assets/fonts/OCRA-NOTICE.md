# OCR-A

OCR-A TrueType font by Matthew Skala (2011-12, 2021), built from a METAFONT
definition originally by Richard B. Wales (1988-89), with later work by Tor
Lillqvist. The bundled file reports "Version 2" (built 2021-03-28).

Source: https://tsukurimashou.org/ocr.php.en
